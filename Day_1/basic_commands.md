                        -- DAY 01 - LINUX BASICS COMMANDS --

- (What I did):
Today I learned some basic Linux commands and practiced them in terminal.

-(Commands):

-pwd
Shows current working directory

-ls
Lists files and folders
Flags:
ls -l → shows file sizes in human readable format
ls -a → shows hidden files

-cd
Change directory
Flags / Variations:
cd .. → go one directory back
cd / → go to root directory

-mkdir
Create a new folder

-touch
Create a new file

-rm
Delete a file

-cat
Display file content

-echo
Print text or write into file

-zcat
View compressed file content

-head
Show first few lines of file

-tail
Show last few lines of file
Flags:
tail -n 5 → show last 5 lines
tail -f → track file updates in real time

-more
View file content page by page

-less
Better way to view large files

Commands do specific tasks
