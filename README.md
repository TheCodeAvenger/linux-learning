# Linux Learning

This repository documents my learning and hands-on practice with Linux system fundamentals.
The focus is on building strong command-line proficiency and understanding core system operations.

---

## Scope of Learning

* Basic & advanced Linux commands
* File system structure and navigation
* File permissions and ownership
* Process and memory management
* Package management

---

## Repository Structure

The repository is organized into notes, daily practice, and hands-on tasks.

* `notes/` → Concept explanations and reference material
* `practice/` → Day-wise command usage and exercises
* `mini-tasks/` → Small practical implementations

Each practice folder includes:

* Commands used
* Explanation of commands
* Terminal outputs (screenshots)
* Key observations

---

## Approach

* Learning by doing (command-line practice)
* Understanding system behavior through real commands
* Documenting outputs and observations
* Maintaining consistency with regular updates
