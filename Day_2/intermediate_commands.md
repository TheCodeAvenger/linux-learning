-- Day 02 - Linux Intermediate Commands --

-(What I did):
Today I learned some intermediate Linux commands and practiced them in terminal.

-(Commands):

1. cp
   Copy files
   Flags:

   * cp -r → copy folders recursively

2. mv
   Move or rename files

3. wc
   Count file content
   Flags:

   * wc -l → count lines
   * wc -w → count words
   * wc -c → count characters

4. ln
   Create hard link

5. ln -s
   Create symbolic (soft) link

6. cut
   Extract parts of text
   Flags:

   * cut -b → extract by bytes

7. tee
   Output to both terminal and file

8. clear
   Clear terminal screen

9. diff
   Compare two files

10. vi
    Open file in editor
